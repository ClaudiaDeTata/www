Grids
The grid system is a responsive layout system, which allows to organize the content into rows and columns. It is based on a 12-col layout. The grid system supports 6 responsive breakpoints, they affects the mentioned bp and all those above it. (sm,md,lg,xl,xxl).
Containers
Class used to provide a responsive fixed-width container. These provides padding and margins, making sure the content is centered and stays within a specified width.
  1. .container = max-width;
  2. .container-(breakpoint) = 100% width until the breakpoint (sm,md,lg,xl,xxl);
  3. .container-fluid = 100% at all breakpoints.
Rows
Rows are wrappers for cols, each one with horizontal padding called Gutter.
Columns
In each row the available columns are 12. The width of the cols is defined for example as col-4 (4 template cols).
Gutters
Gaps between cols (24px or 1.5rem).
